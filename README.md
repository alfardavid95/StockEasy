# StockEasy
-StockEasy
Sistema Web de Control de Inventario para una pequeña empresa

StockEasy es una aplicación web diseñada para ayudar a pequeñas empresas a gestionar su inventario de forma sencilla y organizada. El sistema permite administrar productos, categorías, proveedores, entradas y salidas de inventario, usuarios, roles y reportes básicos.

-Problema que resuelve

Muchas pequeñas empresas manejan su inventario de forma manual mediante cuadernos, hojas de cálculo o registros poco organizados. Esto puede generar errores en las existencias, pérdida de información y dificultad para conocer qué productos entran, salen o tienen bajo stock.

StockEasy busca centralizar esta información en un sistema web que facilite el control de existencias y apoye la toma de decisiones sobre compras, reposición y administración de mercadería.

-Objetivo del proyecto

Desarrollar un sistema web de control de inventario para pequeñas empresas, que permita gestionar productos, categorías, proveedores, entradas y salidas de mercadería, facilitando la consulta de existencias y la identificación de productos con bajo stock.

-Organización de ramas
main: rama principal con versión estable del proyecto.
develop: rama de integración de avances.
feature/productos: desarrollo del módulo de productos.
feature/inventario: desarrollo de entradas, salidas y movimientos.
feature/reportes: desarrollo de reportes.
feature/usuarios: desarrollo de usuarios y roles.
Acuerdo básico de trabajo
Cada integrante debe trabajar en una rama asignada.
No se debe subir código directamente a main.
Los cambios deben integrarse primero en develop.
Antes de hacer merge, se debe revisar que el código funcione correctamente.
Los commits deben tener mensajes claros y relacionados con el cambio realizado.
